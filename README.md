# Healthcare-Analytics
Axon EMR Healthcare is struggling to convert EMR data into meaningful insights due to siloed systems,manual reporting and lack of real-time visibility into clinical data and operational KPI's.
Stakeholders are relying on outdated Excel reports making decision slow and reactive.

To address these issues this project focuses on developing dashboard for Electronic Medical Records (EMR) in Excel,Power BI,Tableau to provide actionable insights for healthcare providers and administrators in order to :

Integrate EMR,billing and operational data

Automate Reporting Process

Build Interactive dashboards for real-time insights

Improve performance tracking accross clinics and departments.
